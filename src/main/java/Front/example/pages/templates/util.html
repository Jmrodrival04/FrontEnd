Utilidades del Sistema

DateUtil

La clase DateUtil proporciona métodos para manejar y formatear fechas.

  • formatDate(LocalDateTime date): Formatea la fecha en una cadena.
  • parseDate(String date): Convierte una cadena en formato de fecha a LocalDateTime.

SecurityUtil

La clase SecurityUtil contiene métodos relacionados con la encriptación y verificación de contraseñas.

  • encryptPassword(String password): Encripta una contraseña.
  • checkPassword(String rawPassword, String encodedPassword): Verifica una contraseña sin encriptar con una encriptada.

AuditUtil

La clase AuditUtil proporciona métodos para registrar eventos de auditoría.

  • logEvent(String action, String performedBy): Crea un log de auditoría con acción y usuario.

StringUtil

La clase StringUtil incluye métodos para manipular cadenas.

  • isNullOrEmpty(String str): Verifica si una cadena es nula o está vacía.
  • capitalizeFirstLetter(String str): Capitaliza la primera letra de una cadena.
