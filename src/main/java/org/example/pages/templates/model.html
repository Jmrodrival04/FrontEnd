Entidades del Sistema

Spell

La entidad Spell representa un hechizo en el sistema.

  • id (Long): Identificador único del hechizo.
  • name (String): Nombre del hechizo.
  • type (String): Tipo de hechizo.
  • effect (String): Descripción del efecto del hechizo.

User

La entidad User representa un usuario del sistema.

  • id (Long): Identificador único del usuario.
  • username (String): Nombre de usuario.
  • role (String): Rol del usuario, como USER o ADMIN.

AuditLog

La entidad AuditLog almacena un registro de auditoría.

  • id (Long): Identificador único del registro de auditoría.
  • action (String): Acción realizada en el sistema.
  • performedBy (String): Usuario que realizó la acción.
  • timestamp (LocalDateTime): Fecha y hora del registro.

Role

La entidad Role define roles de usuario en el sistema.

  • id (Long): Identificador único del rol.
  • roleName (String): Nombre del rol.
