Clases de Lógica de Negocio

MagicEvent

La clase MagicEvent representa un evento mágico en el sistema.

  • description (String): Descripción del evento mágico.
  • timestamp (LocalDateTime): Fecha y hora en que ocurrió el evento.

SpellEffect

La clase SpellEffect define los efectos de un hechizo.

  • effectType (String): Tipo de efecto (ej. daño, curación).
  • intensity (int): Nivel de intensidad del efecto.

UserRole

La clase UserRole gestiona los roles y permisos en el sistema.

  • roleName (String): Nombre del rol.
  • permissions (String[]): Lista de permisos asociados al rol.

AuditRecord

La clase AuditRecord representa un registro de auditoría para eventos específicos.

  • action (String): Acción registrada.
  • performedBy (String): Usuario que realizó la acción.
  • timestamp (LocalDateTime): Fecha y hora del evento.
